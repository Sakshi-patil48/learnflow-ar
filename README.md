# 🚀 LearnFlow AR

LearnFlow AR is an AI-powered educational platform that transforms traditional textbook learning into interactive 3D and AI-assisted experiences. Built with modern web technologies, it combines Google Gemini AI, immersive concepts, and a responsive interface to make learning more engaging, intuitive, and effective.

---

## ✨ Features

- AI-powered learning assistant
- Interactive 3D visualization
- WebXR Real Camera AR View & 3D Interactive Hotspots
- Exploded View Assembly Slider & X-Ray Wireframe Mode
- AI-Powered MCQ Quiz Generator & Instant Answer Verification
- 3D Revision Flashcards Deck with Spaced Repetition & Audio
- Voice Assistant with Text-to-Speech & Speech-to-Text Dictation
- Gamified Student Leaderboard, XP System, Levels & Achievement Badges
- Smart Notes & Printable PDF Summary Exporter
- Admin Management Console (Student Manager, Curriculum & Asset Editor)
- Textbook scanning & NCERT Bookshelf
- Smart chapter explanations
- Responsive UI
- Fast performance
- Modular architecture
- Future-ready design
